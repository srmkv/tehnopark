# Jitsi Excalidraw backend

Simple [Excalidraw] collaboration backend based on [excalidraw-room].

## Building

```bash
npm install
npm run build
```

## Development

```bash
npm run start:dev
```

## Production

```bash
npm start
```

[Excalidraw]: https://excalidraw.com
[excalidraw-room]: https://github.com/excalidraw/excalidraw-room
